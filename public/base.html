Backbone.Events

Events的基本用法

Api: on / off / trigger

先定义一个object, 并使其具有Backbone.Events的特性:

var obj = {}; _.extend(obj, Backbone.Events);

绑定一些事件:

book.on('alert', function(msg) { alert('Triggered event from book Object: ' + msg) }); book.off('alert'); book.trigger('alert', 'trigge event!');

BackBone.Model中对Events的使用

book.on("change:title change:author", ...);

将事件绑定到属性上.这个特性在Backbone.Model中被充分使用 

var Note = Backbone.Model.extend({
	defaults: {
		"title": 'jsgarden',
		"content": 'text....'
	}
});
var note = new Note;

note.on('change:title', function() {
	alert('new data of title: ' + this.get('title'));
});
